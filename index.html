Projects

ARM Consultancy Project

Created an open-source software and data processing model to perform local speaker recognition in near real-time. Trained an advanced convolutional neural network to extract the acoustic feature information from spectrograms of people voices. Developed an algorithm in Python that runs on a Raspberry Pi4 to extract the input voice features using the trained CNN and match the speaker’s input voice to a person in the database. Correctly identified most speakers in roughly 1.3 seconds per inference.

Music Synthesiser

Wrote an embedded software in C++ to make a music synthesiser work. The system was implemented using interrupts and threads to achieve concurrent execution of tasks, such as detecting key presses and generating the output waveform. Used a mutex and queue for thread-safe data sharing. Implemented the ability to send and receive messages to other keyboard modules (when stacked together) using the CAN bus. Advanced features: waveform selection to change the timbre, polyphony to generate multiple tones together & handshaking.

Sound Recognition Device

Notisound is smart sound recognition device that classifies doorbell and fire alarm noises in a house enviroment and sends notification to the end user through an app when the sounds are detected. This embedded device uses a signal-processing algorithm to pre-process the microphone inputs into spectrograms, a convolutional neural network to constantly classify the sounds and a mobile phone app to receive the notifications.

Mars Rover Project

Worked in a team of 6 to design and build an autonomous rover system that could be used in a remote location without direct supervision. The rover is capable to receive movement commands and send status data. The rover sucessfully detects and avoids obstacles in its working area and it is able to build a map and store data of its travelled path. Hardware used: Arduino, DC motors, FPGA, ESP32, camera, SMPS.

CPU Design Project

Worked in a team of 4 to design a general-purpose CPU using Quartus Prime and Verilog HDL, able to execute three algorithms: calculating Fibonacci numbers, calculating pseudo-random integers using a linear congruential generator and traversing a linked list to find an item. The CPU follows a Harvard architecture design in order to implement the desired algorithms.
